# 📚 Web Dev Course Tracker

A simple interactive course tracker to monitor my progress through the **"সহজ সরল সিম্পল"** web development course.

## 🔗 Live Demo

[Click here to view the tracker](https://sajib1915.github.io/Module-Tracker/) 

## ✨ Features

* ✅ Track progress for every lesson across all 5 Milestones
* 📊 Overall progress bar with completion percentage
* 💾 Progress saved in localStorage — no data lost on refresh
* 📱 Responsive design — works on mobile too
* 🗂️ Collapsible milestones and modules for easy navigation

---

## 📁 Project Structure

```
course-tracker/
└── index.html      # Single file app — HTML + CSS + JS
```

---

## 🚀 How to Use Locally

1. Clone or download this repository
2. Open `index.html` in any browser
3. Start ticking off lessons as you complete them!

---

## 🌐 How to Deploy on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch → `/ (root)`
4. Save — your live link will appear in a few seconds
